Detailed Description

template<class CppParamType, class ROSCppParamType>
class SimulinkParameterArrayGetter< CppParamType, ROSCppParamType >

Class for getting array ROS parameters in C++.

This class is used by code generated from the Simulink ROS parameter blocks. Note that the ROSCppParamType template parameter needs to refer to a container type that supports the following operations:

  • resize
  • std::copy std::string (used for string parameters) and std::vector (used for numeric arrays) fall into this category.
