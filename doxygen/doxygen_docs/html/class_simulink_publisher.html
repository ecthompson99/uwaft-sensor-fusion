Detailed Description

template<class MsgType, class BusType>
class SimulinkPublisher< MsgType, BusType >

Class for publishing ROS messages in C++.

This class is used by code generated from the Simulink ROS publisher blocks and is templatized by the ROS message type and Simulink bus type.
