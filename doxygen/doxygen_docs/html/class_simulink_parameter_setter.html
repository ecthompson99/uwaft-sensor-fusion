Detailed Description

template<class CppParamType, class ROSCppParamType>
class SimulinkParameterSetter< CppParamType, ROSCppParamType >

Class for setting ROS parameters in C++.

This class is used by code generated from the Simulink ROS parameter blocks and is templatized by the expected C++ data type for the parameter value.
