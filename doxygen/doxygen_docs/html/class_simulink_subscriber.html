Detailed Description

template<class MsgType, class BusType>
class SimulinkSubscriber< MsgType, BusType >

Class for subscribing to ROS messages in C++.

This class is used by code generated from the Simulink ROS subscriber blocks and is templatized by the ROS message type and Simulink bus type.
