Detailed Description

Signals in message ObstacleStatus.

Values: 00000000 - No Failsafe 00000001 - Low Sun 00000010 - Blur Image 00000100 - Smear Image 00001000 - Partial Transparent Blockage 00010000 - Full Blockage 00100000 - Splashes 01000000 - Out of Calibration 10000000 - Impaired VD

All signal values are as on the CAN bus.
