Utility function to convert strings in a string array from a roscpp message to Simulink bus property.

This function modifies the SL_Info metadata for the individual strings.

Assumptions: 1) The caller is responsible for modifying the SL_Info metadata for the string array itself (e.g., the total number of strings in the string array). 2) numStringsToCopy has already been bounds-checked

Parameters
busProp[out]Fixed-length string array property in Simulink bus
msgProp[in]Fixed-length string array property in roscpp message
warnStatus[in]Handler for warnings during conversion
numStringsToCopy[in]Number of strings to copy

int slros::getNumItemsInFixedSimpleArray ( ArrayType &  array)

Calculate the number of items in a fixed-size simple array.

"Constant size" refers to an array that is declared like "MyType foo[N]" (as opposed to a STL container like vector). "Simple array" means that the size of an array items is completely known at compile time (specifically, a vanilla C struct).

Parameters
arrayReference to the array. Do not pass in a pointer!
Return values
intThe number of items in the array

When converting a roscpp message to a Simulink bus array, the ReceivedLength and CurrentLength values for the Simulink BusInfo struct need to be set appropriately.

This utility function sets
these values and emits a truncation warning if needed.

Assumptions: 1) The caller is responsible for the actual copying of the content to the bus array (note that busProp is a const input)

Parameters
busProp[in]Variable-length array property in Simulink bus
busInfoProp[out]Info property in Simulink bus corresponding to variable-length array
msgProp[in]Variable-length array property in in roscpp message
warnStatus[in]Handler for warnings during conversion
Return values
intThe number of items to be copied (aka. CurrentLength)
