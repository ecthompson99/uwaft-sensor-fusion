template<class T >
bool param_parser::getScalar ( const XmlRpc::XmlRpcValue &  xmlValue,
T &  paramValue 
)

Parse a scalar parameter value.

This function is templatized by the expected data type of the ROS parameter. This function is needed, since the standard ROS C++ parameter parsing does not strictly enforce data type consistency.

Parameters
[in]xmlValueThe value of the parameter as XML-RPC data structure
[out]paramValueThe value of the parameter as scalar data type
Return values
TRUEif parameter with given type was parsed successfully. The value of the parameter will be returned in paramValue.
FALSEif parameter type did not match content in XML-RPC data structure
